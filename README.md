# Speech Therapy Treatment Database
## 言語治療實證方法資料庫

A bilingual (English / Traditional Chinese) evidence-based reference tool for speech-language pathologists covering 40+ treatment approaches across 6 domains:

- **Speech** 言語 — Minimal Pairs, Cycles, DTTC, ReST, NDP3, and more
- **Language** 語言 — Milieu Teaching, Hanen, Narrative Intervention, SCERTS, and more
- **Voice** 嗓音 — LSVT LOUD, SOVT, Resonant Voice, VFE, Accent Method, and more
- **Fluency** 流暢度 — Lidcombe, Palin PCI, Fluency Shaping, ACT, and more
- **Swallowing** 吞咽 — MDTP, Shaker, EMST, Mendelsohn, NMES, CTAR
- **Feeding** 進食 — SOS, Food Chaining, Behavioural Feeding, Neonatal Oral Motor, and more

## Features
- Filter by domain and age group
- Bilingual display (English / Traditional Chinese)
- Evidence ratings (1–5 stars)
- APA 7th edition references with DOIs
- Fully offline — single HTML file + JSON data

## Usage
Open `index.html` in any modern browser. The page loads `data/speech_therapy_db.json` automatically.
